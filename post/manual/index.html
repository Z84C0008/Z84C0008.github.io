PC－80 用户指南

写在前面：这台机器的CPU位置实际上可以连接一个特殊的扩展板 它可以把CPU的所有管脚引出 便于拓展 ／ 更换振荡器 ／ Reset 但是在Type 1 中并未包括 如果使用的是Type 1 跳到：Type - 1 用户指南

由于默认情况下没有显示芯片（因为已经停产而无法购买到全新／散新件）输出会被放到LCD上

如果没有输出，检查ROM是否锁紧／烧录器是否工作正常／烧录器兼容性 / 对比度

1.1 启动

本计算机在您收到时就应该是可以启动的 默认情况下烧录了电子钟程序 只要看到1602屏幕上有输出 即为正常工作 如果没有显示 尝试调整对比度 ／ 检查并口线（杜邦线）

ATTENTION:由于本机没有BIOS电池 电子钟可能并不准确 仅作判断启动是否正常的依据 如果不准确可以忽略

1.2 输入／输出

本机有2个并口 如需串口／PS2 ， 把SIO Expansion （Milestone）和 PS2 Expander 链接到CPU扩展槽

1.3 主板部分说明

主板上有Reset按钮 按下可使 ／RESET 脚位被拉低 可使其重置 如果遇到程序卡死 按下按钮即可 无需插拔电源

关于电源的特殊说明

本机电源包含在一分二USB Type B电源线中 为了避免对CPU造成干扰 请尽可能使用原装电源 不然可能出现一些莫名其妙对问题（亲测会有）

2.1 元件型号

CPU：Z84C0010-XXX PIO：任何一种Z80 PIO芯片（有一些可能管脚定义不同 即使封装相同） RAM：HM628128 ROM：AT28C256E－25DM Screen：LCD1602A

2.2 关于本机

没事干搓的 比较简陋的Z80机 比不上当年的高端计算机 没有ROM Bank ／ 磁盘支持 但是可以拓展出来 理论上找到当年的板卡插上也是可以使用的 但是仅限于理论 因为这台机器采用了新的10MHz Z80芯片 可能需要换到2MHz ／ 4MHz才能支持当年的板卡 但是切换的功能已经在CPU拓展板上提供 （通过跳线切换）

跳线1｜跳线2｜跳线3 1 0 0 = 10Mhz 0 1 0 = 4Mhz 0 0 1 = 2Mhz

	ATTENTION: 在Ver1.0之前的板子上三个跳线实际上是拨码开关 （排针缺货）
